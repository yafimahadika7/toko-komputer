# Store Admin (PHP + Tailwind, v0-style UI)

## Run (XAMPP / Apache)
1. Copy folder to `htdocs/toko-komputer`
2. Import `database.sql` into MySQL
3. Edit `config/db.php` if needed
4. Open:
   - http://localhost/toko-komputer/public/index.php?route=login

## Menus (match reference)
- Dashboard
- Products
- Categories
- Transactions
- Customers
- Suppliers
